        content: |
          Dengan GroupDocs.Metadata for Java API, Metadata Properties dari format file yang didukung dapat diambil dengan cukup efisien. Kode untuk melakukannya cukup sederhana dan mudah. Berikut adalah contoh yang menunjukkan betapa mudahnya mengambil properti metadata dari file MP3 menggunakan Java:
          
          ```java
          try (Mp3Format mp3Format = new Mp3Format("D:\\sample.mp3")) 
          {
          // baca album di ID3 v1
          System.out.printf("Album: %", mp3Format.getId3v1Properties().getAlbum());

          // baca judul di ID3 v2
          System.out.printf("Title: %", mp3Format.getId3v2Properties().getTitle());
          }
          ```
