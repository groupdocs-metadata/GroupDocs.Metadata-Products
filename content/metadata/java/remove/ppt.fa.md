steps:
    enable: true
    title: "مراحل حذف فراداده به PPT در Java"
    content: |
      [GroupDocs.Metadata](https://products.groupdocs.com/metadata/java/) با اجرای چند مرحله آسان، برنامه‌نویسان Java می‌توانند اطلاعات فراداده فایل‌های PPT را از داخل برنامه‌های خود حذف کنند.
      
      1. فایل PPT را برای به روز رسانی بارگیری کنید.
      2. یک گزاره جستجو را به روش RemoveProperties منتقل کنید.
      3. تعداد املاکی که واقعا حذف شده اند را بررسی کنید.
      4. تغییرات را ذخیره کنید.

      content: |
        ```java {style=abap}
        // ابرداده سند PPT را پاک کنید
        try (Metadata metadata = new Metadata("input.ppt");
        {
            // همه نام‌های افراد مشارکت‌کننده در ایجاد فایل را حذف کنید
            // یک ویژگی سفارشی با نام مشخص شده را حذف کنید
            int affected = metadata.removeProperties(
                new FallsIntoCategorySpecification(Tags.getPerson()).
                or(new WithNameSpecification("CustomProperty")));
            
            System.out.println(String.format("Affected properties: %s", affected));
            
            // فایل پاک شده را ذخیره کنید
            metadata.save("output.ppt");
        }
        
        ```
